PS1 Generator
=============

A simple webpage to help generate a PS1 environment variable for your unix shell.

Installation instructions:
--------------------------

1) Install lein and cljsbuild plugin.
2) Run "lein cljsbuild once".
3) Go to index.html in your browser.
